


Image Classifier Application
Overview
This is an image classification application that utilizes a Natural Language Processing (NLP) based machine learning model. The model is designed for image classification and has been implemented using the Keras and TensorFlow frameworks.

Requirements
Make sure you have the following dependencies installed:

numpy
tensorflow
matplotlib
PIL
